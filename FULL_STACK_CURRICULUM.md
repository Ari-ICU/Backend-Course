# 🚀 Full-Stack Developer Curriculum (4 Months)
**Beginner to Industry-Ready Specialist**

- **Duration:** 16 Weeks (4 Months)
- **Commitment:** Monday – Friday, 1 hour/day
- **Stack:** HTML, CSS, JavaScript, React, PHP, Laravel, PostgreSQL
- **Goal:** Build and deploy a multi-vendor E-commerce system.

---

## 🗓️ Phase 1: Frontend Foundations (Month 1)

### Week 1: The Skeleton (HTML5)
*Objective: Build semantic and accessible web structures.*
- **Mon:** Web Internals (How browsers work, HTTP), HTML Basics (`tags`, `elements`).
- **Tue:** Document Structure (Head, Body, Metadata, Semantic HTML).
- **Wed:** Forms & Validation (Inputs, Types, Selects, Attributes).
- **Thu:** Multimedia & Links (Img, Video, Audio, Anchor tags).
- **Fri:** Tables & Lists + **Project: Personal Portfolio Skeleton.**
- **Assignment:** Create a multi-page resume using only HTML.

### Week 2: The Skin (CSS3)
*Objective: Style websites with modern design principles.*
- **Mon:** Selectors, Specificity, and the Box Model (Padding, Margin, Border).
- **Tue:** Typography & Colors (Web fonts, HSL/RGB, Contrast accessibility).
- **Wed:** Flexbox Mastery (Alignment, Distribution, Order).
- **Thu:** Grid Layouts (Tracks, Areas, Fractional units).
- **Fri:** Responsive Design (Media Queries, Mobile-first approach).
- **Assignment:** Style the Portfolio from Week 1 using CSS Flex and Grid.

### Week 3: Interaction Part 1 (Modern JS Basics)
*Objective: Understand logic, data types, and DOM manipulation.*
- **Mon:** Variables (const/let), Scoping, and Data Types.
- **Tue:** Control Flow (If/Else, Switch, Ternary Operators).
- **Wed:** Loops (For, While, Map, Filter, Reduce).
- **Thu:** Functions (Arrow functions, Declarations, Parameters).
- **Fri:** DOM Selection and Event Listeners (Click, Input, Submit).
- **Assignment:** Build a "Calculator" or "Tip Splitter."

### Week 4: Interaction Part 2 (Asynchronous JS & APIs)
*Objective: Fetching data and handling local storage.*
- **Mon:** Objects, Arrays, and Destructuring.
- **Tue:** `this` keyword, Prototypes vs Classes.
- **Wed:** Promises & Async/Await (Handling the "Wait").
- **Thu:** Fetch API & JSON (Consuming public APIs).
- **Fri:** Browser Storage (localStorage/sessionStorage).
- **Assignment:** Build a **Weather App** fetching data from OpenWeatherMap.

---

## 🏗️ Phase 2: Modern Frontend with React (Month 2)

### Week 5: React Core Concepts
*Objective: Transition from imperative to declarative UI.*
- **Mon:** Introduction to Vite & JSX.
- **Tue:** Components (Functional) & Props.
- **Wed:** State Management (`useState`) & Lifting State Up.
- **Thu:** The Component Lifecycle (`useEffect`).
- **Fri:** Conditional Rendering & Lists.
- **Assignment:** Build a **Todo Application** with persistent storage.

### Week 6: Advanced React & Hooks
*Objective: Professional state and performance management.*
- **Mon:** Complex State (`useReducer`).
- **Tue:** Form Handling in React (Controlled vs Uncontrolled).
- **Wed:** Tailwind CSS with React (Rapid Styling).
- **Thu:** React Router (Navigation & Dynamic Imports).
- **Fri:** Context API (Global State).
- **Assignment:** Create a **Product Catalog** with category filters and search.

### Week 7: Building the E-commerce UI
*Objective: Create the storefront components.*
- **Mon:** UI Patterns: Modals, Drawers, and Toasts.
- **Tue:** Shopping Cart Logic (Add/Remove/Update Quantity).
- **Wed:** Checkout Workflow (Multi-step forms).
- **Thu:** Animations using Framer Motion.
- **Fri:** Data Fetching with TanStack Query (React Query).
- **Assignment:** Build a functional **Shopping Cart** with a checkout summary.

### Week 8: Frontend Mastery & Milestone
*Objective: Polish, Optimization, and Deploy.*
- **Mon:** React Performance Optimization (memo, useMemo).
- **Tue:** Unit Testing basics (Vitest/Jest).
- **Wed:** Error Boundaries & Accessibility (A11y).
- **Thu:** Deployment (Vercel/Netlify).
- **Fri:** **MILESTONE: Final E-commerce Frontend Review.**

---

## 🐘 Phase 3: Backend & Database (Month 3)

### Week 9: PHP & PostgreSQL Foundations
*Objective: Master server-side logic and relational data.*
- **Mon:** PHP Syntax, Types, and Superglobals (`$_GET`, `$_POST`).
- **Tue:** Object-Oriented PHP (Classes, Inheritance, Interfaces).
- **Wed:** Intro to PostgreSQL (Installation, Tables, Data Types).
- **Thu:** Basic SQL (SELECT, INSERT, UPDATE, DELETE).
- **Fri:** Joining Tables & Query Optimization.
- **Assignment:** Create a simple PHP script that connects to PG and stores User data.

### Week 10: Laravel Essentials (The Modern PHP Way)
*Objective: Learn the most popular PHP framework.*
- **Mon:** Installation & Architecture (MVC).
- **Tue:** Routing & Controllers.
- **Wed:** Blade Templating Engine.
- **Thu:** Validations & Requests.
- **Fri:** Authentication (Laravel Breeze/Sanctum).
- **Assignment:** Build a **Blog Engine** where users can post and edit articles.

### Week 11: Laravel Models & Eloquent ORM
*Objective: Manage complex data relationships effortlessly.*
- **Mon:** Migrations & Seeders.
- **Tue:** One-to-One and One-to-Many Relationships.
- **Wed:** Many-to-Many Relationships (Pivot tables).
- **Thu:** Query Scopes & Mutators.
- **Fri:** API Development (Returning JSON, API Resources).
- **Assignment:** Extend the Blog to include Categories (Many-to-Many) and User Profiles.

### Week 12: Backend for E-commerce
*Objective: Build the core business logic of a shop.*
- **Mon:** Designing the E-commerce Schema (Products, Orders, Items).
- **Tue:** File Storage (Handling Image Uploads for Products).
- **Wed:** Middleware & Security (CORS, Sanity checks).
- **Thu:** Background Jobs & Queues (Sending Email Notifications).
- **Fri:** **MILESTONE: Functional Backend API for Products/Auth.**

---

## 🏁 Phase 4: Integration & Capstone (Month 4)

### Week 13: Full-Stack Integration
*Objective: Connecting React to Laravel.*
- **Mon:** Setting up JWT/Sanctum Authentication on React.
- **Tue:** Private Routes and Protected API endpoints.
- **Wed:** Handling Global Loading & Error states.
- **Thu:** CORS & Environment Variables (.env).
- **Fri:** Optimistic UI Updates.

### Week 14: Payments & Advanced Features
*Objective: Finishing the E-commerce brain.*
- **Mon:** Stripe/PayPal API Integration.
- **Tue:** Order Processing & Inventory Management.
- **Wed:** User Reviews & Ratings System.
- **Thu:** Admin Dashboard UI (Laravel Nova or Custom Filament).
- **Fri:** Search Functionality (Meilisearch or simple SQL LIKE).

### Week 15: Testing & Security
*Objective: Making the system production-ready.*
- **Mon:** Integration Testing (Laravel Pest/PHPUnit).
- **Tue:** E2E Testing with Playwright/Cypress.
- **Wed:** SQL Injection & XSS Protection.
- **Thu:** Performance Profiling (Laravel Telescope).
- **Fri:** SSL, Nginx Config & Env Security.

### Week 16: Deployment & Final Presentation
*Objective: Launching to the world.*
- **Mon:** Dockerizing the App (Optional but recommended).
- **Tue:** VPS Setup (DigitalOcean/AWS).
- **Wed:** CI/CD Pipelines (Github Actions).
- **Thu:** Final Bug Squashing & UI Polish.
- **Fri:** **GRADUATION: Present E-commerce System!**

---

## 🏆 Final Project: "NexusMarket"
**Features required:**
1.  **Identity:** Secure User & Admin login/registration.
2.  **Commerce:** Product search, sorting, filtering, and pagination.
3.  **Checkout:** Functional Stripe Payment integration.
4.  **Admin:** Portal to add products, view orders, and manage users.
5.  **Notifications:** Email receipts and order status updates.
